## Creative Brief & Prompt

### Project overview
- Single Next.js (App Router) repo.
- Two landing pages:
  - `/` → personal landing page for Jordan Christley.
  - `/emerald-tech` → Emerald Technology Consulting landing page.
- Deploy via GitHub Pages at `jordanchristley.com` with path routing.

### Brand goals
- Minimalist layout with bold emphasis on the most important line.
- Visual-forward sections (gradients, cards, logo cloud, motion where tasteful).
- Modern, premium, creative feel.
- Emerald page should feel like a dark, polished gem: deep green + glow.

### Typography
- Primary: Inter (clean, modern, reliable).
- Display: Space Grotesk (bold, creative, tech-forward).
- Fall back to system fonts on Apple/Google/Windows.

### Personal landing page (`/`)
- Hero copy: “Hi, I’m Jordan. Developer & entrepreneur.”
- Supporting line: sleek, modern, confident.
- CTA buttons:
  - Primary: “Hire me” → link to `/emerald-tech`
  - Secondary: “Get in touch”
- Minimal sections:
  - Short highlights of what I do
  - A few visual cards for “Now building / Fast ships / Design-led / Founder energy”
- Contact form:
  - Fields: Name, Email, Message
  - Submit text: “Send a note”
  - Note: “Email not wired yet—ready to connect later.”

### Emerald Technology Consulting (`/emerald-tech`)
- Hero copy: “Emerald Technology Consulting” + “A premium technology studio…”
- Primary CTA: “Start a project”
- Secondary CTA: “Back to main site”
- Sections:
  - Service cards (Product engineering, Experience design, AI integration,
    Platform hardening, Launch velocity)
  - Engagement model / delivery approach
  - Logo cloud (large, visual)
- Contact form:
  - Fields: Name, Email, Company, Message
  - Submit text: “Request a consult”
  - Note: “Email delivery intentionally not wired yet.”

### Logo cloud list
React, Next.js, TypeScript, JavaScript, Node.js, AWS, Python, PostgreSQL,
MySQL, MongoDB, Supabase, Prisma, Tailwind, OpenAI, Anthropic, Heroku,
Express, Docker, Vercel, Git, Figma.

### Visual treatment
- Dark base with emerald glow.
- Subtle gradients, blurred orbs, and glassy cards.
- Use 1–2 strong accents, avoid clutter.
- Prefer large typography, clear hierarchy, short lines.

### Inspiration notes
- **hey.milo.gg**: immersive, minimalist, bold hero energy with motion.
- **roshan-sahu.com**: large typography, grid/card layouts, portfolio-style
  sections, strong CTA.
